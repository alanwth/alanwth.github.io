Bristol Hub Teenage code club volunteering

The motivations for becoming a volunteer for the Code Club project are simple, not only can I learn more about python myself, but I can also address the skill gap between the future employment market and the teenagers with disadvantaged backgrounds. As a volunteer of this project, I had to familiarize myself with the fundamental of Python. To do this, I have followed the instructions as stated on the code club website and have prepared a range of material ready to teach the teenagers at the community centre.

The first few weeks of teaching will consist of the fundamental syntax of Python as well as the logic between simple function such as an if and else if loop. To towards the end of the project, we are hoping the pupils would be able to create a simple game using pygame and different libraries if needed.
througout this project, I am also hoping to solidify my understanding of the different concepts in Python, only by truly understanding it myself, would I be able to explain to the studnets how different function works and behave.

Week 1

The basic of Python

The aim of the first week is to get the students to familiarize themselves with Python and some of the most basic functions such as print() and input(). The students were tasked to create a programme to ask for input from the user and to conduct simple mathematical operations with them.

Week 2

Rock, Paper, Scissors

After the first session, we have adjusted the material used for the second session. While keeping the session enjoyable for the students, we were also aimed to introduce some concepts that are more advanced. The primary objective was to get them to know and understand the different types of loops and applying to a simple game of rock, paper, scissors.

Tic Tac Toe

Upon the first two sessions, the students have gained an introductory view on the Python programming language and are now more capable of building more sophisticated programmes. We have decided to introduces Tic Tac Toe to the class as it involves more complicated logic and concepts. Nevertheless, they were able to learn about list and if statements> with this exercise.
