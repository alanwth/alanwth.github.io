Bristol Hub Teenage code club volunteering

The motivations for becoming a volunteer for the Code Club project are simple, not only can I learn more about pyton myself, but I can also address the skill gap between the future employment market and the teenagers with disadvantage backgrounds. As a volunteer of this project, I had to familize myself with the fundamental of Pyhton. To do this I have followed the instrustion as stated on the code club website and have perpare a range of material ready to teach the teenagers at the youth centre. The first few week of teaching will be consisting of the fundamental syntac of Python as well as the logic between simple function such as a if and else if loop. To towards the end of the projrct (week 10), we are hoping the pupils would be able to create a simple game using pygame and differnt libraies if needed.

By commiting to this project, no only have I solidify my understanding of Python and have got my hands on to trying out differernt projects, I have also got a sense of acheviement wehn I see the pupil who are finally able to write different programmes by themselves, seeing what their creatively can bring has also made myself feel pround.
